Fascículo Técnico: Pre-Cálculo

1. Álgebra Operacional

  • Diferencia de Cuadrados: $a^2 - b^2 = (a - b)(a + b)$
  • Trinomio al Cuadrado: $(a \pm b)^2 = a^2 \pm 2ab + b^2$
  • Ecuación Cuadrática: $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
  • Racionalización: Técnica para eliminar raíces del denominador multiplicando por el conjugado.

2. Funciones y Restricciones

  • Dominio Real: Conjunto de valores de $x$ donde la función existe.
  • Restricción Fraccionaria: Si $f(x) = \frac{1}{Q(x)}$, entonces $Q(x) \neq 0$.
  • Restricción Radical (índice par): Si $f(x) = \sqrt{P(x)}$, entonces $P(x) \geq 0$.

3. Valor Absoluto e Inecuaciones

  • El valor absoluto $|x|$ es la distancia al origen (siempre $\geq 0$).
  • Inecuación Abierta: $|x - a| < b \iff -b < x - a < b$.

4. Logaritmos y Exponentes

  • Leyes de Exponentes: $(x^a)^b = x^{ab}$ ; $x^a \cdot x^b = x^{a+b}$ ; $x^{-a} = 1/x^a$.
  • Definición Log: $\log_b(x) = y \iff b^y = x$.
  • Propiedades Naturales: $\ln(a) + \ln(b) = \ln(a \cdot b)$ ; $\ln(1) = 0$.

5. Trigonometría Fundamental

  • Identidad Pitagórica: $\sin^2(\theta) + \cos^2(\theta) = 1$.
  • Cociente: $\tan(\theta) = \frac{\sin(\theta)}{\cos(\theta)}$.
  • Ángulos Notables: $\tan(45^\circ) = 1$ ; $\cos(60^\circ) = 1/2$.
  • Conversión: $\pi \text{ rad} = 180^\circ$ (Sistema Sexagesimal).
